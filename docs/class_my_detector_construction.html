Detailed Description

Defines the detector geometry, materials, and sensitive detectors.

The MyDetectorConstruction class inherits from G4VUserDetectorConstruction and is used to define and build the detector setup for the simulation. This includes defining the materials, constructing the physical volumes, and assigning sensitive detectors and fields.
