Detailed Description

Manages the initialization of primary generator and run action.

This class inherits from G4VUserActionInitialization and defines the user-defined actions needed for the Geant4 simulation, including the primary generator and run action.

Builds the user-defined actions for the simulation.

This method is responsible for creating and registering user-defined actions to the Geant4 framework. It sets the following:

  • A primary generator action (MyPrimaryGenerator).
  • A run action (MyRunAction).
