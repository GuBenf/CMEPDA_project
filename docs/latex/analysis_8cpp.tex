\doxysubsection{Detailed Description}
Codice per analizzare i dati simulati tramite Geant@4. 

Restituisce un plot degli intervalli di tempo fra muone rivelato e positrone di decadimento rilevato con fit esponenziale decrescente. Quando si chiama la funzione analysis bisogna specificare il nome del file dei dati e chi sta facendo l\textquotesingle{}analisi ("{}g"{} per Guglielmo, "{}a"{} per Alberto), così da selezionare automaticamente il path per la cartella. Se non si seleziona uno dei due autori noti è possibile fornire direttamente il percorso della cartella come secondo argomento.
